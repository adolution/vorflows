01 BaseFlow aufsetzen
Fehlermeldung bei der ersten Eingabe? Hier klicken

Du hast VS Code und Claude Code wie im Tutorial installiert, bekommst aber bei der ersten Eingabe eine rote Fehlermeldung (z. B. „claude is not recognized" oder „You must run claude to review the updated terms")? Dann muss Claude nur einmalig freigeschaltet werden, das dauert zwei Minuten. Schau dir bei Bedarf nochmal den ersten Teil des Videos bis zur fertigen Installation an:

Nur falls die Fehlermeldung erscheint, führ diese Schritte aus:

  1. Terminal öffnen. In VS Code oben im Menü auf Terminal → Neues Terminal. Unten erscheint ein Eingabefeld.
  2. Claude installieren. Kopier die Zeile für dein Gerät, klick ins Terminal, füg sie ein, drück Enter. Danach das Terminal einmal schließen und neu öffnen.
    Windowsirm https://claude.ai/install.ps1 | iex
    Maccurl -fsSL https://claude.ai/install.sh | bash
  3. Anmelden & bestätigen. Tipp claude ins Terminal, Enter. Dein Browser öffnet sich zum Anmelden, mit deinem Konto einloggen, zurück ins Fenster. Claude zeigt einmal die Nutzungsbedingungen → mit Enter bestätigen. Fertig.

Im Browser-Login oder in der Claude-Desktop-App zu bestätigen reicht nicht — die Bestätigung muss in Claude Code selbst passieren (Schritt 4). Mehr unter Troubleshooting.
