// 07Meta Pixel & Meta Conversions API

Diese Website nutzt den Meta Pixel sowie die Meta Conversions API der Meta Platforms Ireland Limited, 4 Grand Canal Square, Dublin 2, Irland.

Meta Pixel (clientseitig)

Der Meta Pixel ist ein JavaScript-Code-Snippet, das in Ihrem Browser Conversion-Ereignisse erfasst (z. B. Seitenaufruf, Klick auf einen Button, Formularabsendung). Die dabei erhobenen Daten werden an Meta übertragen und können zur Messung der Wirksamkeit unserer Werbeanzeigen sowie zur Erstellung von Custom Audiences für Retargeting-Kampagnen genutzt werden.

Meta Conversions API (serverseitig)

Ergänzend zum Pixel nutzen wir die Meta Conversions API, um Conversion-Ereignisse direkt von unserem Server an Meta zu übermitteln. Dabei werden Daten wie gehashte E-Mail-Adressen, IP-Adressen oder Browserkennung (User-Agent) in anonymisierter/gehashter Form übertragen. Eine Identifizierung einzelner Personen durch Meta ist durch das Hashing nicht direkt möglich.

Verarbeitete Datenkategorien

  • IP-Adresse
  • Browser-Informationen (User-Agent)
  • Besuchte URLs und Conversion-Ereignisse
  • Gehashte E-Mail-Adresse (nur bei Formularabsendung, SHA-256)
  • Klick-ID (fbclid) aus Meta-Anzeigen

Datenübertragung in die USA

Meta Platforms Inc. (USA) kann auf die Daten zugreifen. Meta ist nach dem EU-US Data Privacy Framework zertifiziert. Zusätzlich werden Standarddatenschutzklauseln vereinbart.

Rechtsgrundlage & Widerspruch

Rechtsgrundlage ist Ihre Einwilligung gemäß Art. 6 Abs. 1 lit. a DSGVO. Sie können Ihre Einwilligung jederzeit über unser Cookie-Banner widerrufen. Zusätzlich können Sie der personalisierten Werbung unter facebook.com/adpreferences widersprechen.

Datenschutzerklärung Meta: facebook.com/privacy/policy

// 08Cookies

Unsere Website verwendet Cookies. Cookies sind kleine Textdateien, die auf Ihrem Endgerät gespeichert werden. Wir unterscheiden zwischen:

  • Technisch notwendige Cookies — für den Betrieb der Website erforderlich, keine Einwilligung nötig
  • Analyse-Cookies (Google Analytics, Hotjar) — nur mit Ihrer Einwilligung
  • Marketing-Cookies (Meta Pixel) — nur mit Ihrer Einwilligung

Sie können Ihre Einwilligung jederzeit über unser Cookie-Banner widerrufen oder Cookies in Ihrem Browser deaktivieren. Beachten Sie, dass die Deaktivierung bestimmter Cookies die Funktionalität der Website einschränken kann.
