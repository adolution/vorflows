## Design Context

### Users

**Shopify founders, post-hype, technical-curious operators.**

- Already running a shop with real revenue. Not hobbyists.
- Stuck on app subscriptions, agency dependencies, or theme limits.
- Treat AI as leverage, not toy. Will smell hype instantly.
- Want to do the work themselves — not delegate, not "managed service".
- Comfortable enough with code to recognize when it's serious. Knows what a `.liquid` file is.
- Browsing context: comparison-shopping vs. hiring an agency or buying yet another app. Skeptical default.
- Job to be done: *"Show me the mechanism. Convince me I can run my shop on senior-dev level without an agency or a 15-app stack."*

### Brand Personality

**Three words: senior, direct, crafted.**

- **Senior** — speaks like someone who has actually shipped Shopify themes, not a marketer dressed as one.
- **Direct** — consultant tone. Names what works and what doesn't. No padding.
- **Crafted** — premium feel through restraint and detail, not loud effects.

Emotional goal: founder feels *"finally someone who knows what they're talking about — and trusts me to handle it."* Quiet confidence, not pep talk. No urgency tricks, no fake scarcity.

Voice forbiddens: "revolutionary", "game-changer", "unlock", "10x", "AI magic", any SaaS bro-speak. German primary. Sharp, declarative sentences.

### Aesthetic Direction

**Editorial premium with terracotta warmth. Mixed-theme: warm light surfaces with dark sections for hero / showcase / module reveals.**

Locked tokens (from `index.html`):

- **Light bg:** `#FAFAF7` (warm off-white, paper-leaning)
- **Dark bg:** `#0F0F0F` (near-black, slight warmth — never pure `#000`)
- **Ink:** `#1A1A1A` on light, `#EAEAE5` on dark
- **Secondary ink:** `#5C5C5C` light / `#8A8A82` dark
- **Borders:** `#E5E3DD` (soft) / `#C7C4BB` (strong) / `#2A2A2A` (dark mode)
- **Accent:** `#C8633E` terracotta (single hero color — no second accent, no rainbow)
- **Accent hover:** `#B05530`
- **Accent soft tint:** `rgba(200, 99, 62, 0.08)`
- **Diff add / remove:** `#2F6F4F` / `#8B3A3A` (used in code/diff blocks)

Type system (locked):

- **Display:** `Fraunces` (serif, opsz 9–144). Tight tracking `-0.02em` to `-0.03em` on large headings. Italic variant for editorial accent.
- **Body:** `Inter` 400/500/600. Body 17px / line-height 1.6. `font-feature-settings: "ss01", "cv11"`.
- **Mono:** `JetBrains Mono` 400/500. For code, file paths, diffs, technical labels.
- **Handwritten:** `Caveat` 500/600. Sparingly — annotation/margin-note feel only.
- **Meta label:** Inter 500, uppercase, tracking `0.08em`, 12px.

Brand assets present in `brand_assets/`:

- Logos: paper-style mark (`07a-paper-*`), wordmark light + dark (`10d.1-wordmark*`), favicon set
- Box renders (3D, tonal): `base`, `seo`, `clarity`, `family` — all 1×1 webp, transparent + solid variants. **Locked geometry & palette** (see memory `feedback_box_design_locked.md`).

Anti-references — explicitly NOT this energy:

- Cheap-course / scam-coach landing pages (red arrows, neon yellow CTAs, fake-timer popups, founder-with-Lambo hero)
- Generic vibe-coded SaaS sites (purple-blue gradient on dark, glass cards, indigo-500 buttons, hero-metric-template)
- Anything that screams "no serious business behind this"
- AI-slop fingerprints: cyan-on-dark, glow accents, identical card grids, bouncy easing, monospace as "tech vibe" shorthand

Reference vibes (none provided — design from first principles, leaning editorial / craft-magazine / boutique-studio over SaaS-template).

### Motion Stance

- **Default:** purposeful motion only. One signature moment per section, not constant micro-twitches.
- **Easing:** exponential / quart / quint out only. No bounce, no elastic, no overshoot — they read cheap.
- **Animate only `transform` and `opacity`.** Never layout properties.
- **Durations:** 100–200ms for feedback, 300–500ms for state changes, 500–800ms for entrances. Exits ~75% of entrance.
- **Reduced motion:** **reduce stance** (not full kill). Under `prefers-reduced-motion: reduce`: drop transforms/parallax/scroll-driven reveals, keep short opacity fades (<150ms). Premium feel preserved, no nausea triggers.
- **Forbidden:** `transition-all`, scroll-jacking, autoplay video that competes for attention, decorative animation without functional reason.

### Design Principles

1. **Mechanism over marketing.** Show the workflow (file export → Claude project → duplicate theme → review → publish). Diagrams, real file names, real code beats abstract "AI-powered" claims.
2. **Restraint signals premium.** Generous whitespace, single accent color, one display font, sparse animation. Loud = cheap in this audience.
3. **Editorial composition, not template grids.** Asymmetry, varied spacing rhythm, mixed type sizes. No identical 3-column "feature card" rows.
4. **Terracotta is precious.** Single accent, used surgically — CTAs, key word emphasis, one diagram highlight per section. Never gradient, never decorative wash.
5. **Mixed theme tells the story.** Light = explanation, calm, reading. Dark = hero, module reveal, showcase, code. Transitions between them are part of the rhythm.
6. **Every interactive surface earns its state.** Hover, focus-visible, active — all three, all the time. Cursor changes. Focus rings visible (terracotta on light, off-white on dark).
7. **Respect the reader's time.** No redundant headers. No intro that restates the heading. No section without a job.
